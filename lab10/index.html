    Wat is DOM?

    DOM also known as The COument Object Model is programming for web documents thatrepresents the page structure as a tree of the object such as element, attribute or text.

    It serves as a bridge for scripting languages like JavaScript which allows access to different structures and styles in a web page.

    You can think of DOM as a blueprint that is organized in a hierarchical way.
