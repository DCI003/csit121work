How is the DOM Used? (with examples)

JavaScript interacts with the DOM through its API (Application Programming Interface). Here are some common ways the DOM is used with JavaScript:

Accessing Elements

You can select HTML elements using various methods:

  • document.getElementById('myId'): Selects an element by its unique ID.
  • document.querySelector('.myClass'): Selects the first element with the specified class.
  • document.querySelectorAll('p'): Selects all paragraph elements.

                // Example: Getting an element by its ID
                const heading = document.getElementById('main-heading');
                console.log(heading.textContent); // Output: Welcome to the DOM!

Modifying Content

You can change the text content or HTML content of an element:


                // Example: Changing the text content
                heading.textContent = 'DOM Manipulation!';

                // Example: Changing the HTML content
                const myDiv = document.getElementById('myDiv');
                myDiv.innerHTML = 'This is new content.';

Changing Styles

You can dynamically alter the CSS styles of elements:


                // Example: Changing the background color
                heading.style.backgroundColor = 'lightblue';
                heading.style.color = 'white';

Adding and Removing Elements

You can create new elements and add or remove them from the DOM:


                // Example: Creating a new paragraph element
                const newParagraph = document.createElement('p');
                newParagraph.textContent = 'This paragraph was added by JavaScript.';

                // Example: Appending the new paragraph to the body
                document.body.appendChild(newParagraph);

                // Example: Removing an element
                // newParagraph.remove();

Handling Events

You can attach event listeners to elements to respond to user interactions:


                // Example: Adding an event listener to a button
                const myButton = document.getElementById('myButton');
                myButton.addEventListener('click', function() {
                    alert('Button clicked!');
                });
